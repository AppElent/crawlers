# kaas_nl_v2 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Create `kaas_nl_v2.ts` — a fixed rewrite of `kaas_nl.ts` that uses `PlaywrightCrawler` for JS-rendered FacetWP pagination and correct CSS selectors from the live site.

**Architecture:** Two-phase Crawlee crawl (LIST → DETAIL). LIST pages use Playwright to wait for the product grid, collect detail links, and enqueue the next FacetWP page. DETAIL pages extract title, segment, flavor, and description. Seen URLs are persisted in a `KeyValueStore` across runs.

**Tech Stack:** TypeScript (ESM), Crawlee 3.x, `PlaywrightCrawler`, Playwright (peer dep), Biome (lint/format)

### Task 4: Implement the LIST handler

**Files:**
- Modify: `kaas_nl_v2.ts`

The LIST handler waits for the product grid, enqueues all detail links, then enqueues the next page if there is one.

- [ ] **Step 1: Add the crawler with LIST handler after the `isNew` function**

Add this after the `await requestQueue.addRequest(...)` line, still inside `run()`:

```ts
    // const proxyConfiguration = new ProxyConfiguration({ proxyUrls: ['http://...'] });

    const crawler = new PlaywrightCrawler({
        requestQueue,

        async requestHandler(ctx: PlaywrightCrawlingContext) {
            const { request, page, log } = ctx;
            const label = request.label as Label;

            if (label === 'LIST') {
                log.info(`LIST: ${request.url}`);

                await page.waitForSelector('.cat-cheese');

                // Collect all detail URLs from this page
                const detailUrls = await page.$$eval(
                    '.cat-cheese > a',
                    (els) => els.map((el) => (el as HTMLAnchorElement).href),
                );

                for (const url of detailUrls) {
                    await requestQueue.addRequest({ url, label: 'DETAIL' });
                }

                log.info(`Enqueued ${detailUrls.length} detail URLs`);

                // Enqueue next page if pagination button exists
                const nextPage = await page.$('a.facetwp-page.next');
                if (nextPage) {
                    const dataPage = await nextPage.getAttribute('data-page');
                    if (dataPage) {
                        const nextUrl = `https://www.kaas.nl/kazen/?fwp_paged=${dataPage}`;
                        await requestQueue.addRequest({
                            url: nextUrl,
                            label: 'LIST',
                            uniqueKey: nextUrl,
                        });
                        log.info(`Enqueued next page: ${nextUrl}`);
                    }
                }
            }
        },
    });
```

### Task 5: Implement the DETAIL handler

            if (label === 'DETAIL') {
                const url = request.url;

                if (!(await isNew(url))) {
                    log.info(`SKIP: ${url}`);
                    return;
                }

                log.info(`DETAIL: ${url}`);

                const [title, segment, flavor, description] = await Promise.all([
                    page.$eval('h1.title', (el) => el.textContent?.trim() ?? '').catch(() => ''),
                    page.$eval('h6.subtitle', (el) => el.textContent?.trim() ?? '').catch(() => ''),

                ]);

                const product: Product = { url, title, segment, flavor, description };
                await dataset.pushData(product);
                log.info(`Saved: ${title}`);
            }
